# Project Brief

**Project:** Idle Bricks 3D
**Created:** 2025-11-29

## Purpose
Idle Bricks 3D is a 3D idle breakout-style game that combines automatic ball physics with upgrades and progression to create a lightweight, replayable web experience.

## Scope
- Core gameplay (balls, bricks, scoring)
- Upgrade systems (damage, speed, balls)
- 3D rendering via React Three Fiber
- Lightweight web delivery via Vite

## Key Goals
- Fun, idle progression loop
- Smooth 3D rendering on common devices
- Clear upgrade and economy feedback
